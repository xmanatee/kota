KOTA already has failure and approval alerts, but it still lacks a cleaner operator-facing digest / attention surface.

Explore concise notifications or summaries for things like:
- repeated workflow failures
- budget pressure
- long-running stalled work
- significant completed work that deserves review

This should avoid spam and should complement, not duplicate, the existing approval/failure alerts.
