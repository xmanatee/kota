KOTA has memory modules, but the autonomous workflows do not keep a concise durable memory of recurring lessons, known pitfalls, stable repo knowledge, or repeated failure patterns.

Explore a repo-local memory surface that helps the autonomous workflows avoid re-learning the same things over and over, while staying auditable and concise.

Good solutions should avoid turning memory into a second task queue or a stale doc dump.
