# Inbox

This state is for newly captured ideas that have not been triaged yet.

- Keep entries concise.
- Rough captures are allowed here if they are still understandable.
- Use the same filename convention as other tasks: `task-<slug>.md`.
- Frontmatter is optional here. A short plain-text note is valid if it is understandable.
- If you do add frontmatter, keep it lightweight. Full task normalization happens during triage.
- Move items out quickly once they are understood.
- Do not treat this directory as a long-term backlog.
