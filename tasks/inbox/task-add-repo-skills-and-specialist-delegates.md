KOTA has workflows and ad hoc delegation, but it does not have a reusable repo-scoped skill/delegate surface comparable to Claude Code skills/subagents, Codex skills, or OpenClaw skills.

Explore a project-defined capability system that can package:
- instructions
- supporting files/scripts
- optional manual invocation
- automatic invocation hints
- scoped tools/model/memory for specialist delegates

This should help explorer, builder, and improver call better reusable specialists instead of embedding everything in one prompt or hardcoded workflow.
