# Inbox

This directory is for quick captures that have not been sorted yet.

- Keep entries concise.
- Rough captures are allowed here if they are still understandable.
- Use filename convention `task-<slug>.md` for durable captures, but do not overthink it.
- Frontmatter is optional here. A short plain-text note is valid if it is understandable.
- Full task normalization happens only if something graduates into `data/tasks/`.
- Do not treat this directory as a long-term backlog or work queue.
- When an entry graduates, preserve its source, urgency, and owner wording in
  the destination artifact. Do not smooth away the reason the capture exists.
