Structured knowledge entries as markdown files with YAML front matter.
Use for research findings, decisions with rationale, reference material, and plans.
Supports types, tags, status, and full-text search.
Prefer knowledge over memory for substantial, structured entries.

Completed builder and improver runs automatically produce `run-insight` entries
tagged with `run:<runId>`, `workflow:<name>`, and `task:<taskId>`. Query these
to learn what past runs built, which files they changed, and what patterns
emerged. Example: search for type `run-insight` or a specific task tag.
