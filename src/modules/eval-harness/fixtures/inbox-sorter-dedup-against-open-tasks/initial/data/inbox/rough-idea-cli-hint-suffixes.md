# Rough capture: hint suffixes on kota CLI error messages

Thinking the kota CLI could append a short hint to each error message
pointing at the most likely next step the operator should try. Right
now errors surface the failure but not the recovery path, which slows
troubleshooting for anyone who is not already deep in the code.
